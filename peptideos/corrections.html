Corrections Log

Public record of all corrections, clarifications, and updates

Every correction is logged here with a date, description, and reason. This log is the transparency mechanism. If you find an error, report it. If we fix it, it goes here.

The log is sorted reverse-chronological (newest first). Historic versions of all pages are archived on GitHub.

How to Report an Error

Found a mistake? Send a message to contact@opensourcemed.info or file an issue on GitHub.

Corrections for factual errors (citing the wrong source, mischaracterizing evidence tier) are prioritized. Style and phrasing suggestions are noted but deprioritized. The log is not a comment thread.
