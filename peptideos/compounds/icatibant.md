# Icatibant (Firazyr)

Bradykinin B2-receptor antagonist approved for acute hereditary angioedema attacks.

**Mechanism:** Blocks bradykinin B2 receptor → prevents bradykinin-mediated vascular permeability increase → reduces swelling.

## Evidence Summary

Time to improvement ~4 hours (icatibant) vs. ~12–24 hours placebo in hereditary angioedema attacks.

## Safety Profile (S2)

Injection site reactions (~80%, transient erythema/warmth), rare transient hypotension.
