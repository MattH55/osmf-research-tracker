# Cagrilintide (Investigational)

## MANDATORY STANDING NOTICE

**This compound is investigational. It is not approved in any jurisdiction. There is no lawful supply. Any material sold under this name is of unverified origin and neither manufactured nor tested to pharmaceutical standards.**

---

Amylin receptor agonist in Phase 3 development (sponsor: Novo Nordisk, Ziltivekimab collaboration).

**Mechanism:** Amylin receptor agonism → centrally mediated glucose-dependent insulin secretion + glucagon suppression + gastric emptying delay.

## Evidence Summary

### Type 2 Diabetes (E2)
Phase 2b RCT (SUSTAIN-AL, N~500): 3.0% additional HbA1c reduction when added to semaglutide. Phase 3 initiated (SUSTAIN-C1 series).

### Weight Loss (E3)
Phase 2 preliminary data shows weight loss benefit with dual GLP-1 + amylin mechanism; limited reporting.

## Safety Profile (S3)

**Adverse Events:**
- Nausea/vomiting (60%, dose-limiting; ~15% discontinuation)
- Pancreatitis signal (preclinical amylin agonism; one possible case Phase 2b, causality unclear)
- Hypoglycemia (theoretical with dual GLP-1/amylin; not observed in Phase 2b with careful titration)

## Trial Status

- SUSTAIN-C1/C2 (T2DM): Phase 3, ongoing
- SUSTAIN-AL obesity arm: Phase 2, ongoing

Expected approval timeline: 2027–2028 (estimated from Phase 3 initiation).

## What Is Unknown

1. **Phase 3 efficacy.** SUSTAIN-C1 results pending (expected 2027)
2. **Pancreatitis risk.** Preclinical signal; one possible case in Phase 2b; Phase 3 safety database will clarify
3. **Nausea at scale.** Phase 2b showed 60%; Phase 3 dose escalation may worsen tolerability
4. **Hypoglycemia risk.** Theoretical with dual mechanism; not yet observed in controlled trials
