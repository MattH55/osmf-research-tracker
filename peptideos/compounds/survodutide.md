# Survodutide (Investigational)

## MANDATORY STANDING NOTICE

**This compound is investigational. It is not approved in any jurisdiction. There is no lawful supply. Any material sold under this name is of unverified origin and neither manufactured nor tested to pharmaceutical standards.**

---

GLP-1/GCG dual agonist in Phase 2 clinical development (sponsor: Roche/Carmot).

**Mechanism:** Glucagon-like peptide-1 (GLP-1) receptor + glucagon (GCG) receptor co-agonism → dual glucose-dependent insulin secretion + gluconeogenesis inhibition + weight loss.

## Evidence Summary

### Type 2 Diabetes (E3)
Early Phase 2 data reported; efficacy and tolerability profiles vs. tirzepatide / retatrutide not yet characterized.

### Weight Loss (E4)
Phase 2 ongoing; no human efficacy data available.

## Safety Profile (S4—Moderate Uncertainty)

**Expected:**
- Nausea/GI (40–60%, dose-limiting, based on dual mechanism)
- Gallbladder/biliary events (GCG agonism emergent signal; retatrutide Phase 3 shows signal; survodutide data limited)

**Data Gap:** Limited human exposure; Phase 3 will clarify safety profile.

## Trial Status

- SURPASS-D1 (T2DM): Phase 2, recruiting/ongoing
- TRIUMPH-S1 (obesity): Phase 2, enrollment status unknown

Approval timeline: Unknown. No public regulatory guidance from Roche.

## What Is Unknown

1. **Efficacy in T2DM.** Phase 2 data not yet reported; Phase 3 not initiated
2. **Efficacy in obesity.** Phase 2 ongoing
3. **GCG-specific safety.** Biliary/gallbladder events (emergent signal in retatrutide); survodutide risk profile unclear
4. **Pancreatitis risk.** Unknown with dual GLP-1/GCG mechanism
5. **Nausea tolerability.** Expected severe; comparative data vs. tirzepatide pending
