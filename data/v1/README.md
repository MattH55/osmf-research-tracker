# PAIS Cohort Database — v1.x frozen snapshot

These files are the v1.1 release, preserved at stable URLs because the dataset may have been cited.
The live database has since migrated to the v2 heterogeneous (Observation-based) schema; see ../pais-cohorts-index.json.

- `pais-cohorts-index.json` — v1.1 compiled index (Finding + SymptomFinding model)
- `pais-cohorts.csv`, `pais-symptom-findings.csv` — v1.1 bulk exports
- `pais-cohort.schema.v1.json` — the v1 schema
